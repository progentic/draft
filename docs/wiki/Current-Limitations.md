# Current Limitations

DRAFT is still pre-release software. The writing workspace is currently
available, but most planned product workflows are not yet available from the
interface.

## Documents

- You cannot create, open, save, close, or reopen a document file from the
  workspace.
- Autosave and recovery are not available.
- Reloading or closing the application discards editor changes.

## Research And Citations

- There is no visible reference library.
- External metadata search is currently unavailable from the workspace.
- You cannot insert citations or generate a bibliography from controls.
- You cannot import or process a PDF from the workspace.

## Analysis And Text Review

- Automated source analysis is currently unavailable.
- Grammar, clarity, tone, cohesion, and voice findings are not shown in the
  workspace.
- DRAFT does not silently rewrite document text.

## Formatting And Export

- The toolbar formats the current editor selection, but complete APA, MLA, or
  Chicago conformance is not claimed.
- Formatting review currently checks heading structure and citation-style
  declarations only. Findings are transient and citation mismatches are
  inspect-only.
- DRAFT does not automatically repair formatting or certify complete
  style-manual compliance.
- You cannot export a DOCX file from the workspace.
- PDF export is currently unavailable. DRAFT has deferred that work until its
  rendering policy and implementation boundary are defined and verified.

This wording describes the current state, not a permanent product limitation.

## Downloads And Packaging

- No versioned DRAFT release has been published yet.
- A supported installer, signing process, notarization policy, update channel,
  and release package are not currently available.
- macOS on Apple Silicon is the planned initial release platform.

## Privacy And External Access

- The current workspace does not run automated analysis or metadata searches.
- DRAFT does not currently include product telemetry.
- Opening research links and sending text to external services are not currently
  available from the workspace.
- Offline mode is process-local and resets to online when DRAFT restarts.
- DRAFT does not monitor operating-system connectivity, retry or queue external
  work, or persist the mode.
- API keys, external-service providers, credential prompts, and credential
  settings are not currently available in the workspace.
