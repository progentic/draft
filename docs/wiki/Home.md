# DRAFT

DRAFT is a local-first desktop workspace for research-heavy writing.

The current pre-release workspace lets you write and format one document,
navigate its headings, view live document statistics, and confirm that the
desktop core is available. Your editor content is temporary at this stage.

## Start Here

- [Using the workspace](Workspace.md)
- [Troubleshooting](Troubleshooting.md)
- [Current limitations](Current-Limitations.md)

## Before You Begin

DRAFT has not published a versioned release yet. The current workspace does not
save or reopen documents from the interface. Do not use it as the only copy of
important work.
